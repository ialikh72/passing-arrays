# Array Printer & Sum Calculator (C#)

This is a simple C# console application that demonstrates:

- Passing arrays to methods  
- Iterating over arrays using `foreach` and `for` loops  
- Calculating the sum of array elements  
- Printing output to the console  

## 📌 What the Program Does

The program:

1. Prints each element of an integer array  
2. Calculates and displays the sum of the elements
